#  Web-Based To-Do Application

A simple and efficient web-based To-Do application built using **Spring Boot** for the backend and **JSP (JavaServer Pages)** for the frontend. This application allows users to manage daily tasks with ease by adding, updating, and deleting to-do items.

---

##  Features

*  Add new tasks
*  Update existing tasks
*  Delete tasks
*  View all tasks in a structured list
*  Persistent storage using backend integration
*  Web-based interface using JSP

---

##  Tech Stack

### Backend

* Java
* Spring Boot
* Spring MVC

### Frontend

* JSP (JavaServer Pages)
* HTML, CSS

### Tools & Build

* Maven
* Apache Tomcat (embedded)

##  Installation & Setup

### Prerequisites

* Java 8 or higher
* Maven
* Git

### Steps to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/Rahul234u/Web-Based-Todo-Application.git
   ```

2. Navigate to the project directory:

   ```bash
   cd todo-app
   ```

3. Build the project:

   ```bash
   mvn clean install
   ```

4. Run the application:

   ```bash
   mvn spring-boot:run
   ```

5. Open your browser and visit:

   ```
   http://localhost:8080
   ```
